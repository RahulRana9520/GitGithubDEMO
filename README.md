# Git & GitHub Learning Website

A modern interactive website to learn Git and GitHub, built with React (frontend) and Node.js/Express (backend).

## Features
- What is Git & GitHub (clear, visual explanations)
- Basic workflow (Working Directory → Staging → Repository → GitHub)
- Live demo/code simulation for Git commands
- Branching, collaboration, conflict resolution, rollback, and pro tips
- Mini hackathon simulation challenge
- Responsive, visually appealing design

## Getting Started

### 1. Install dependencies

Open two terminals:

#### Frontend
```
cd client
npm install
```

#### Backend
```
cd server
npm install
```

### 2. Run the app

#### Start backend (from /server):
```
npm start
```

#### Start frontend (from /client):
```
npm start
```

- The frontend runs on http://localhost:3000
- The backend runs on http://localhost:5000

## Customization
- Edit content in `client/src/data/sections.js` to update learning modules.
- Style components in `client/src/components/` and CSS files.
